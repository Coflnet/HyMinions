# HyMinions

Hypixel Skyblock - Minions Calculator and Events Timer. Know your most profitable minions! Tailor made for each profile, fully accurate.

## Installation

Requires Node.js and npm

```bash
npm install
```

or

```bash
yarn
```

## Usage

```bash
npm run start
```

or

```bash
yarn start
```
